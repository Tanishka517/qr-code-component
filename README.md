# Frontend Mentor - QR code component

![Design preview for the QR code component coding challenge](./preview.jpg)
A clean card layout that displays:
- A QR code image
- A heading
- A short description text

## Welcome! 👋

Thanks for checking out this front-end coding challenge.

This is a simple responsive QR code component built using HTML and CSS.  
It is based on a Frontend Mentor challenge.

---


---

## 🛠️ Built With

- HTML5
- CSS3 (Flexbox)
- Responsive design techniques

---

## 📱 Features

- Fully centered layout
- Responsive card design
- Clean and minimal UI
- Mobile-friendly design

---

## 🎯 What I Learned

- Flexbox centering
- Responsive design using `max-width`
- Image responsiveness (`width: 100%`)
- Basic UI structuring

## 🚀 How to Use

1. Clone the repository
2. Open `index.html` in your browser
3. Or use Live Server in VS Code
